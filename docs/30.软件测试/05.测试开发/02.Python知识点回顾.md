## 数据类型

### 1.字符串

-   定义：‘’，”“，‘’‘’‘’

-   json：

    -   json.loads()：json字符串 ---> 字典

    -   json.dumps()：字典 ---> json

        

### 2.列表与元组

-   元组 ---> 数据不允许修改

-   列表 ---> 数据可以更改

    

### 3.字典

-   定义：{‘键’：值}

-   键是唯一的，值可以不唯一

    

### 4.集合（set）

-   定义：一个无序的不重复的元素序列

-   可以使用大括号{}或者set()函数创建集合

    ::: warning

    创建空集合必须使用`set()`而不是{}

    :::

-   去重：

    ```python
    list = [1,2,3,3]
    
    print(set(list1))   
    ```

-   交集：

    ```python
    set1 = {1,2,3,4}
    set2 = {4,5,6,7}
    
    print(set1 & set2)
    ```
