CSS属性大全

# CSS 属性大全

# 一、动画属性(Animation)

属性 描述
animation (opens new window) 指定基于关键帧的动画。
animation-delay (opens new window) 指定动画何时开始。
animation-direction (opens new window) 指定动画是否应在交替的循环中反向播放。
animation-duration (opens new window) 指定动画完成一个周期应花费的秒数或毫秒数。
animation-fill-mode (opens new window) 指定CSS动画在执行之前和之后应如何将样式应用于其目标。
animation-iteration-count (opens new window) 指定在停止之前动画循环应播放的次数。
animation-name (opens new window) 指定@keyframes应应用于所选元素的已定义动画的名称。
animation-play-state (opens new window) 指定动画是运行还是暂停。
animation-timing-function (opens new window) 指定CSS动画在每个周期内应如何进行。

# 二、背景属性(Background)

属性 描述
background (opens new window) 在一个声明中定义各种背景属性。
background-attachment (opens new window) 指定背景图像是在视口中固定还是滚动。
background-clip (opens new window) 指定背景的绘制区域。
background-color (opens new window) 定义元素的背景色。
background-image (opens new window) 定义元素的背景图像。
background-origin (opens new window) 指定背景图像的定位区域。
background-position (opens new window) 定义背景图像的原点。
background-repeat (opens new window) 指定是否/如何平铺背景图像。
background-size (opens new window) 指定背景图像的大小。

# 三、边框属性(Border)

属性 描述
border (opens new window) 设置元素边框所有四个侧面的宽度，样式和颜色。
border-bottom (opens new window) 设置元素底部边框的宽度，样式和颜色。
border-bottom-color (opens new window) 设置元素底部边框的颜色。
border-bottom-left-radius (opens new window) 定义元素的左下边界角的形状。
border-bottom-right-radius (opens new window) 定义元素右下边界的形状。
border-bottom-style (opens new window) 设置元素底部边框的样式。
border-bottom-width (opens new window) 设置元素底部边框的宽度。
border-color (opens new window) 设置元素所有四个侧面的边框颜色。
border-image (opens new window) 指定如何使用图像代替边框样式。
border-image-outset (opens new window) 指定边框图像区域超出边框超出范围的数量。
border-image-repeat (opens new window) 指定图像边框应重复，四舍五入还是拉伸。
border-image-slice (opens new window) 指定图像边框的向内偏移。
border-image-source (opens new window) 指定要用作边框的图像的位置。
border-image-width (opens new window) 指定图像边框的宽度。
border-left (opens new window) 设置元素左边框的宽度，样式和颜色。
border-left-color (opens new window) 设置元素的左边框的颜色。
border-left-style (opens new window) 设置元素左边框的样式。
border-left-width (opens new window) 设置元素左边框的宽度。
border-radius (opens new window) 定义元素边界角的形状。
border-right (opens new window) 设置元素右边框的宽度，样式和颜色。
border-right-color (opens new window) 设置元素右边框的颜色。
border-right-style (opens new window) 设置元素右边框的样式。
border-right-width (opens new window) 设置元素右边框的宽度。
border-style (opens new window) 在元素的所有四个面上设置边框的样式。
border-top (opens new window) 设置元素顶部边框的宽度，样式和颜色。
border-top-color (opens new window) 设置元素顶部边框的颜色。
border-top-left-radius (opens new window) 定义元素左上角的形状。
border-top-right-radius (opens new window) 定义元素的右上边界角的形状。
border-top-style (opens new window) 设置元素顶部边框的样式。
border-top-width (opens new window) 设置元素顶部边框的宽度。
border-width (opens new window) 设置元素所有四个侧面的边框宽度。

# 四、颜色属性(Color)

属性 描述
color (opens new window) 指定元素文本的颜色。
opacity (opens new window) 指定元素的透明度。

# 五、尺寸属性(Size)

属性 描述
height (opens new window) 指定元素的高度。
max-height (opens new window) 指定元素的最大高度。
max-width (opens new window) 指定元素的最大宽度。
min-height (opens new window) 指定元素的最小高度。
min-width (opens new window) 指定元素的最小宽度。
width (opens new window) 指定元素的宽度。

# 六、内容属性(Content)

属性 描述
content (opens new window) 插入生成的内容。
quotes (opens new window) 指定嵌入引号的引号。
counter-reset (opens new window) 创建或重置一个或多个计数器。
counter-increment (opens new window) 递增一个或多个计数器值。

# 七、弹性Flex盒子布局(Flex)

属性 描述
align-content (opens new window) 指定伸缩容器中伸缩容器的项目的对齐方式。
align-items (opens new window) 为flex容器中的项目指定默认对齐方式。
align-self (opens new window) 指定弹性容器中选定项目的对齐方式。
flex (opens new window) 指定弹性长度的分量。
flex-basis (opens new window) 指定弹性项目的初始主要尺寸。
flex-direction (opens new window) 指定弹性项目的方向。
flex-flow (opens new window) flex-direction和flex-wrap属性的简写属性。
flex-grow (opens new window) 指定弹性项目相对于弹性容器内其他项目的增长方式。
flex-shrink (opens new window) 指定flex项目相对于flex容器内其他项目的收缩方式。
flex-wrap (opens new window) 指定是否应包装柔性物品。
justify-content (opens new window) 指定在解决了任何弹性长度和自动页边距之后，弹性项目如何沿弹性容器的主轴对齐。
order (opens new window) 指定弹性项目在弹性容器中的显示和布局顺序。

# 八、字体属性(Fonts)

属性 描述
font (opens new window) 在一个声明中定义各种字体属性。
font-family (opens new window) 定义元素的字体列表。
font-size (opens new window) 定义文本的字体大小。
font-size-adjust (opens new window) 发生字体回退时，保留文本的可读性。
font-stretch (opens new window) 从字体中选择一个普通的，压缩的或扩展的字体。
font-style (opens new window) 定义文本的字体样式。
font-variant (opens new window) 指定字体变体。
font-weight (opens new window) 指定文本的字体粗细。

# 九、列表项属性(Lists)

属性 描述
list-style (opens new window) 定义列表和列表元素的显示样式。
list-style-image (opens new window) 指定用作列表项标记的图像。
list-style-position (opens new window) 指定列表项标记的位置。
list-style-type (opens new window) 指定列表项的标记样式。

# 十、边距属性(Margin)

属性 描述
margin (opens new window) 在元素的所有四个面上设置边距。
margin-bottom (opens new window) 设置元素的底边距。
margin-left (opens new window) 设置元素的左边距。
margin-right (opens new window) 设置元素的右边距。
margin-top (opens new window) 设置元素的上边距。

# 十一、多列布局属性(Column)

属性 描述
column-count (opens new window) 指定多列元素中的列数。
column-fill (opens new window) 指定如何填充列。
column-gap (opens new window) 指定多列元素中各列之间的间隔。
column-rule (opens new window) 指定在多列元素的每一列之间绘制的直线或“规则”。
column-rule-color (opens new window) 指定在多列布局中的列之间绘制的规则的颜色。
column-rule-style (opens new window) 指定在多列布局中的列之间绘制的规则的样式。
column-rule-width (opens new window) 指定在多列布局中的列之间绘制的规则的宽度。
column-span (opens new window) 指定元素在多列布局中跨越多少列。
column-width (opens new window) 指定多列元素中列的最佳宽度。
columns (opens new window) 用于设置column-width和column-count属性的简写属性。

# 十二、轮廓属性(Outline)

属性 描述
outline (opens new window) 设置元素轮廓的所有四个边的宽度，样式和颜色。
outline-color (opens new window) 设置轮廓的颜色。
outline-offset (opens new window) 设置轮廓和元素边框之间的空间。
outline-style (opens new window) 设置轮廓样式。
outline-width (opens new window) 设置轮廓的宽度。

# 十三、填充属性(Padding)

属性 描述
padding (opens new window) 在元素的所有四个面上设置填充。
padding-bottom (opens new window) 将填充设置为元素的底侧。
padding-left (opens new window) 将填充设置为元素的左侧。
padding-right (opens new window) 将填充设置为元素的右侧。
padding-top (opens new window) 将填充设置为元素的顶部。

# 十四、打印属性(Print)

属性 描述
page-break-after (opens new window) 在元素之后插入分页符。
page-break-before (opens new window) 在元素之前插入分页符。
page-break-inside (opens new window) 在元素内插入分页符。

# 十五、表属性(Table)

属性 描述
border-collapse (opens new window) 指定是连接还是分隔表格单元格边界。
border-spacing (opens new window) 设置相邻表格单元格的边界之间的间距。
caption-side (opens new window) 指定表格标题的位置。
empty-cells (opens new window) 显示或隐藏空表单元格的边框和背景。
table-layout (opens new window) 指定表布局算法。

# 十六、文字属性(Text)

属性 描述
direction (opens new window) 定义文本方向/书写方向。
tab-size (opens new window) 指定制表符的长度。
text-align (opens new window) 设置内联内容的水平对齐方式。
text-align-last (opens new window) 指定当text-alignis 时如何对齐块的最后一行或强制换行符之前的行justify。
text-decoration (opens new window) 指定添加到文本的装饰。
text-decoration-color (opens new window) 指定的颜色text-decoration-line。
text-decoration-line (opens new window) 指定将哪种线条装饰添加到元素。
text-decoration-style (opens new window) 指定text-decoration-line属性指定的线条样式
text-indent (opens new window) 缩进文本的第一行。
text-justify (opens new window) 指定当text-align属性设置为时要使用的对正方法justify。
text-overflow (opens new window) 指定当文本内容溢出块容器时将如何显示。
text-shadow (opens new window) 将一个或多个阴影应用于元素的文本内容。
text-transform (opens new window) 转换文本的大小写。
line-height (opens new window) 设置文本行之间的高度。
vertical-align (opens new window) 设置元素相对于当前文本基线的垂直位置。
letter-spacing (opens new window) 设置字母之间的额外间距。
word-spacing (opens new window) 设置单词之间的间距。
white-space (opens new window) 指定如何处理元素内的空白。
word-break (opens new window) 指定如何在单词内换行。
word-wrap (opens new window) 指定在内容超出其容器边界时是否中断单词。

# 十七、转换属性(Transform)

属性 描述
backface-visibility (opens new window) 指定当面对用户时，转换后的元素的“背面”是否可见。
perspective (opens new window) 定义从中查看对象的所有子元素的透视图。
perspective-origin (opens new window) 定义透视图属性的原点（3D空间的消失点）。
transform (opens new window) 将2D或3D变换应用于元素。
transform-origin (opens new window) 定义元素的变换原点。
transform-style (opens new window) 指定如何在3D空间中渲染嵌套元素。

# 十八、过渡属性(Transition)

属性 描述
transition (opens new window) 定义元素的两种状态之间的过渡。
transition-delay (opens new window) 指定过渡效果何时开始。
transition-duration (opens new window) 指定过渡效果要花费的秒数或毫秒数。
transition-property (opens new window) 指定应将过渡效果应用到的CSS属性的名称。
transition-timing-function (opens new window) 指定过渡效果的速度曲线。

# 十九、视觉格式属性

属性 描述
display (opens new window) 指定元素在屏幕上的显示方式。
position (opens new window) 指定如何定位元素。
top (opens new window) 指定所定位元素的上边缘的位置。
right (opens new window) 指定所定位元素的右边缘的位置。
bottom (opens new window) 指定所定位元素底边的位置。
left (opens new window) 指定定位元素左边缘的位置。
float (opens new window) 指定一个框是否应该浮动。
clear (opens new window) 指定相对于浮动元素的元素位置。
z-index (opens new window) 指定定位元素的分层或堆叠顺序。
overflow (opens new window) 指定对溢出元素框的内容的处理。
overflow-x (opens new window) 指定当内容超出元素内容区域的宽度时如何管理内容。
overflow-y (opens new window) 指定当内容超出元素内容区域的高度时如何管理内容。
resize (opens new window) 指定元素是否可由用户调整大小。
clip (opens new window) 定义裁剪区域。
visibility (opens new window) 指定一个元素是否可见。
cursor (opens new window) 指定游标的类型。
box-shadow (opens new window) 将一个或多个阴影应用于元素的框。
box-sizing (opens new window) 更改默认的CSS框模型。
