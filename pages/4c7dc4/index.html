# Vue 实战

# 一、 环境配置

# 1. 安装 vscode

vscode 下载 (opens new window)

# 2. nodejs 安装（安装稳定版）

nodejs 下载 (opens new window)

# 3. npm 命令 更换为淘宝源

npm install -g cnpm --registry=https://registry.npm.taobao.org
1

# 4. 安装 Vue 框架

cnpm install -g @vue/cli@3.12.1
1

# 二、 项目

# 1. 新建项目

将 cmd 路径切换到需要创建项目的文件夹下，使用下面的方法创建项目

vue create web_demo
1
  1. 上下调整， 选择 自定义 创建项目

  1. 上下调整，空格选择，回车确认

  1. 直接 回车，选择 默认 路由（history）

  1. 直接 回车 ， 选择 默认 样式器（dart-sass）

  2. 检查模式 ，选择 ESLint + Prettier

  3. 直接回车， 选择默认 触发检查时机

    1. 配置文件放置位置， 选择 放在 In package.json

    2. 选择 N ， 不保存配置

    3. 有 如下提示，表示安装成功

# 出现错误时的解决办法：

以 管理员 的方式运行 windows PowerShell

输入命令 ： set-ExecutionPolicy RemoteSigned

完成。

# 2. 启动项目

  1. 使用 cd web_demo 进入项目
  2. 使用 npm run serve 启动项目

有如下提示表示启动成功：

# 3. 停止项目 安装 粒子特效背景

按两次 ctrl+c 停止项目

在 终端 使用以下命令安装 粒子特效背景

npm install vue-particles --save-dev
1

# 引用 粒子特效

import VueParticles from 'vue-particles';
Vue.use(VueParticles);
1
2

配置特效

<template>
    <div>
        <vue-particles
          color="#556"
          :particleOpacity="0.7"
          :particlesNumber="80"
          shapeType="polygon"
          :particleSize="2"
          linesColor="#555"
          :linesWidth="1"
          :lineLinked="true"
          :lineOpacity="0.4"
          :linesDistance="150"
          :moveSpeed="4"
          :hoverEffect="true"
          hoverMode="grab"
          :clickEffect="true"
          clickMode="push"
          class="lizi"
        >
    </vue-particles>
    </div>
</template>

# 4. 安装 element-ui

在 终端 使用以下命令安装 element-ui

npm i element-ui -S
1

# 引用 element-ui

import ElementUI from "element-ui";
import 'element-ui/lib/theme-chalk/index.css';
Vue.use(ElementUI);
1
2
3

# 5.安装 axios

可以用来做接口请求

在 终端 使用以下命令安装 axios

npm install axios -S
1

# 引用 axios

import axios from 'axios';
Vue.prototype.$axios = axios;
axios.defaults.baseURL = '/api';    // 使用代理，防止接口被拦截报错
axios.defaults.withCredentials = true;	// 绑定获得的cookie
1

# 6. 安装 js-cookie

可以自定义 cookie

在 终端 使用以下命令安装 js-cookie

npm install js-cookie
1

# 引用 js-cookie

import Cookies from 'js-cookie';
Vue.prototype.$cookies = Cookies;
