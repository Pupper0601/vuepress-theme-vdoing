HTML常用标签

# HTML 常用标签

# 一、基础

标签 描述
<!DOCTYPE> (opens new window) 定义文档类型。
<html> (opens new window) 定义一个 HTML 文档
<title> (opens new window) 为文档定义一个标题
<body> (opens new window) 定义文档的主体
<h1> to <h6> (opens new window) 定义 HTML 标题
<p> (opens new window) 定义一个段落
<br/> (opens new window) 定义简单的折行。
<hr> (opens new window) 定义水平线。
<!--...--> (opens new window) 定义一个注释

# 二、格式

标签 描述
<acronym> (opens new window) HTML5不再支持。 定义只取首字母的缩写。
<abbr> (opens new window) 定义一个缩写。
<address> (opens new window) 定义文档作者或拥有者的联系信息。
<b> (opens new window) 定义粗体文本。
<bdi> (opens new window)New 允许您设置一段文本，使其脱离其父元素的文本方向设置。
<bdo> (opens new window) 定义文本的方向。
<big> (opens new window) HTML5不再支持。 定义大号文本。
<blockquote> (opens new window) 定义块引用。
<center> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。定义居中文本。
<cite> (opens new window) 定义引用(citation)。
<code> (opens new window) 定义计算机代码文本。
<del> (opens new window) 定义被删除文本。
<dfn> (opens new window) 定义定义项目。
<em> (opens new window) 定义强调文本。
<font> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。 定义文本的字体、尺寸和颜色
<i> (opens new window) 定义斜体文本。
<ins> (opens new window) 定义被插入文本。
<kbd> (opens new window) 定义键盘文本。
<mark> (opens new window)New 定义带有记号的文本。
<meter> (opens new window)New 定义度量衡。仅用于已知最大和最小值的度量。
<pre> (opens new window) 定义预格式文本
<progress> (opens new window)New 定义运行中的任务进度（进程）。
<q> (opens new window) 定义短的引用。
<rp> (opens new window)New 定义不支持 ruby 元素的浏览器所显示的内容。
<rt> (opens new window)New 定义字符（中文注音或字符）的解释或发音。
<ruby> (opens new window)New 定义 ruby 注释（中文注音或字符）。
<s> (opens new window) 定义加删除线的文本。
<samp> (opens new window) 定义计算机代码样本。
<small> (opens new window) 定义小号文本。
<strike> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。 定义加删除线的文本。
<strong> (opens new window) 定义语气更为强烈的强调文本。
<sub> (opens new window) 定义下标文本。
<sup> (opens new window) 定义上标文本。
<time> (opens new window)New 定义一个日期/时间
<tt> (opens new window) HTML5不再支持。 定义打字机文本。
<u> (opens new window) 定义下划线文本。
<var> (opens new window) 定义文本的变量部分。
<wbr> (opens new window)New 规定在文本中的何处适合添加换行符。

# 三、表单

标签 描述
<form> (opens new window) 定义一个 HTML 表单，用于用户输入。
<input> (opens new window) 定义一个输入控件
<textarea> (opens new window) 定义多行的文本输入控件。
<button> (opens new window) 定义按钮。
<select> (opens new window) 定义选择列表（下拉列表）。
<optgroup> (opens new window) 定义选择列表中相关选项的组合。
<option> (opens new window) 定义选择列表中的选项。
<label> (opens new window) 定义 input 元素的标注。
<fieldset> (opens new window) 定义围绕表单中元素的边框。
<legend> (opens new window) 定义 fieldset 元素的标题。
<datalist> (opens new window)New 规定了 input 元素可能的选项列表。
<keygen> (opens new window)New 规定用于表单的密钥对生成器字段。
<output> (opens new window)New 定义一个计算的结果

# 四、框架

标签 描述
<frame> (opens new window) HTML5不再支持。 定义框架集的窗口或框架。
<frameset> (opens new window) HTML5不再支持。定义框架集。
<noframes> (opens new window) HTML5不再支持。 定义针对不支持框架的用户的替代内容。
<iframe> (opens new window) 定义内联框架。

# 五、图像

标签 描述
<img> (opens new window) 定义图像。
<map> (opens new window) 定义图像映射。
<area> (opens new window) 定义图像地图内部的区域。
<canvas> (opens new window)New 通过脚本（通常是 JavaScript）来绘制图形（比如图表和其他图像）。
<figcaption> (opens new window)New 定义一个 caption for a <figure > element
<figure> (opens new window)New figure 标签用于对元素进行组合。

# 六、Audio/Video

标签 描述
<audio> (opens new window)New 定义声音，比如音乐或其他音频流。
<source> (opens new window)New 定义media元素 (
<track> (opens new window)New 为媒体(
<video> (opens new window)New 定义一个音频或者视频
链接
<a> (opens new window) 定义一个链接
<link> (opens new window) 定义文档与外部资源的关系。
<main> (opens new window) 定义文档的主体部分。
<nav> (opens new window)New 定义导航链接

# 七、列表

标签 描述
<ul> (opens new window) 定义一个无序列表
<ol> (opens new window) 定义一个有序列表
<li> (opens new window) 定义一个列表项
<dir> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。 定义目录列表。
<dl> (opens new window) 定义一个定义列表
<dt> (opens new window) 定义一个定义定义列表中的项目。
<dd> (opens new window) 定义定义列表中项目的描述。
<menu> (opens new window) 定义菜单列表。
<command> (opens new window)New 定义用户可能调用的命令（比如单选按钮、复选框或按钮）。

# 八、表格

标签 描述
<table> (opens new window) 定义一个表格
<caption> (opens new window) 定义表格标题。
<th> (opens new window) 定义表格中的表头单元格。
<tr> (opens new window) 定义表格中的行。
<td> (opens new window) 定义表格中的单元。
<thead> (opens new window) 定义表格中的表头内容。
<tbody> (opens new window) 定义表格中的主体内容。
<tfoot> (opens new window) 定义表格中的表注内容（脚注）。
<col> (opens new window) 定义表格中一个或多个列的属性值。
<colgroup> (opens new window) 定义表格中供格式化的列组。

# 九、样式/节

标签 描述
<style> (opens new window) 定义文档的样式信息。
<div> (opens new window) 定义文档中的节。
<span> (opens new window) 定义文档中的节。
<header> (opens new window)New 定义一个文档头部部分
<footer> (opens new window)New 定义一个文档底部
<section> (opens new window)New 定义了文档的某个区域
<article> (opens new window)New 定义一个文章内容
<aside> (opens new window)New 定义其所处内容之外的内容。
<details> (opens new window)New 定义了用户可见的或者隐藏的需求的补充细节。
<dialog> (opens new window)New 定义一个对话框或者窗口
<summary> (opens new window)New 定义一个可见的标题。 当用户点击标题时会显示出详细信息。

# 十、元信息

标签 描述
<head> (opens new window) 定义关于文档的信息
<meta> (opens new window) 定义关于 HTML 文档的元信息。
<base> (opens new window) 定义页面中所有链接的默认地址或默认目标。
<basefont> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。 定义页面中文本的默认字体、颜色或尺寸。

# 十一、程序

标签 描述
<script> (opens new window) 定义客户端脚本。
<noscript> (opens new window) 定义针对不支持客户端脚本的用户的替代内容。
<applet> (opens new window) HTML5不再支持。 HTML 4.01 已废弃。 定义嵌入的 applet。
<embed> (opens new window)New 定义了一个容器，用来嵌入外部应用或者互动程序（插件）。
<object> (opens new window) 定义嵌入的对象。
<param> (opens new window) 定义对象的参数。
